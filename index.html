Mordekhai Gerine Lumangkun

Perkenalkan nama saya Mordekhai Gerin, biasa di panggil Gerin. Saya berasal dari Kalimantan Timur tepatnya di Kabupaten Kutai Timur, Kota Sangatta. Saya suka editing, apalagi di masa pandemi hampir setiap hari saya melakukan kegiatan ini. Saya pernah terlibat dalam projek pembuatan vidio dokumenter sekolah saya, dan juga projek vidio pelepasan angkatan saya. Selain itu, untuk mengisi waktu kosong saya suka bermain basket. Saya pernah mewakili sekolah saya dalam lomba basket Honda DBL di provinsi saya. Selain itu juga, saya pernah mewakili sekolah saya dalam PASKIBRAKA di tingkat kabupaten tahun 2018.
